# 🌐 LinguaAI — Language Translator

An AI-powered language translation desktop application built during the **CodeAlpha AI Internship (Task 1)**.

---

## 🖥️ About the Project

LinguaAI is a professional desktop translation tool that supports 70+ languages with a clean, modern UI. It uses Google Translate API under the hood and includes text-to-speech functionality to hear your translations out loud.

---

## 🛠️ Languages & Technologies Used

| Technology | Purpose |
|---|---|
| **Python 3** | Core programming language |
| **Tkinter** | Desktop GUI framework |
| **deep-translator** | Google Translate API wrapper |
| **gTTS (Google Text-to-Speech)** | Convert translated text to audio |
| **pygame** | Play the generated audio |
| **threading** | Smooth UI during translation (no freezing) |
| **Canvas (tkinter)** | Decorative header graphics & empty state art |

---

## ✨ Features

- 🌍 Translate between 70+ languages
- 🏳️ Live flag emoji per selected language
- 🔊 Text-to-speech for translated output
- 📋 One-click copy translation
- ⚡ Keyboard shortcut — Ctrl+Enter to translate
- 📊 Character count progress bar (max 5,000)
- 🎨 Professional dark-chrome UI with violet accent
- 💫 Animated loading dots while translating
- 🖼️ Illustrated empty state with globe art

---

## ▶️ How to Run

**1. Clone the repository**
git clone https://github.com/Syedasafiya/LinguaAI-Translator.git
cd LinguaAI-Translator

**2. Install dependencies**
pip install deep-translator gtts pygame

**3. Run the app**
python translator_app.py

> Dependencies auto-install on first run if missing.
