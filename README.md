# LinguaAI — Language Translator

AI-powered translation app built with Python & Google Translate API.

## Features
- 70+ languages
- Text-to-speech (gTTS)
- Copy translation
- Professional UI with flag emojis

## How to Run
pip install deep-translator gtts pygame
python translator_app.py

## Built With
- Python & Tkinter
- deep-translator
- gTTS + pygame
